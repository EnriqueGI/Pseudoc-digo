¿Qué es el Pseudocódigo?

Piensa en el pseudocódigo como una receta de cocina para una computadora. Antes de escribir las instrucciones en un idioma que solo un chef experto (un lenguaje de programación) entendería, escribes los pasos de forma clara y sencilla, para que cualquiera pueda seguir la lógica.

No necesitas seguir reglas estrictas de un lenguaje, pero sí mantener una estructura lógica que te permita, más adelante, traducirlo fácilmente a código real.

Algoritmo SumaDosNumeros
  // 1. Pedir los ingredientes (números)
  Escribir "Dame el primer número"
  Leer num1
  Escribir "Dame el segundo número"
  Leer num2
  
  // 2. Realizar la preparación (cálculo)
  suma <- num1 + num2
  
  // 3. Servir el plato (mostrar resultado)
  Escribir "El resultado es: ", suma
FinAlgoritmo

Variables: Las Cajas de Memoria

Una variable es como una caja con una etiqueta. Dentro de la caja, guardas un dato (un número, una palabra, etc.). La etiqueta es el nombre de la variable, que te permite encontrar y usar el dato guardado cuando lo necesites.

Al declarar Definir edad Como Entero y luego asignar edad <- 25, estás haciendo esto:

edad (Entero)
25
// Creamos las "cajas" y definimos qué guardarán
Definir edad Como Entero
Definir nombre Como Cadena

// Guardamos datos en las cajas
edad <- 25
nombre <- "Ana"

Operadores: Las Herramientas de Cálculo y Comparación

Los operadores son los símbolos que le dicen a la computadora qué hacer con tus variables. Son como las herramientas en una caja: tienes sumadores y restadores (aritméticos), lupas para comparar si algo es más grande o igual (relacionales) y conectores lógicos para tomar decisiones complejas (lógicos).

TipoOperadorEjemplo
Aritméticos
Suma+5 + 3
Relacionales
Igual a==4 == 4
Lógicos
Y (AND)Y(4>2) Y (5<10)

Estructuras de Control: El GPS del Código

Las estructuras de control son como un GPS para tu programa. Le indican qué camino tomar según ciertas condiciones. Permiten que tu código no sea una simple lista de tareas, sino un sistema inteligente que puede tomar decisiones y repetir acciones.
